Menu G5 Examples: Slide-in menu

yxScripts.com

In Menu G5, if part of the menu is placed outside of the browser window and thus one of the menu items will be cut off by the browser border, you can mouseover that item or the item next to it to scroll the outside part in.

Menu G5 supports 4 scrolling options: "both", "x-only", "y-only" or "none", and two offset options for how the menu should scroll in: by one menu item each time, or by some offset each time.

Combining the menu scrolling features and a floating menu, we can make a slide-in menu to save some page space.
