Menu G5 Examples: Using images in menu [3]

yxScripts.com

 

Menu G5 introduces image tiles to make a better background for the menu pads.

The nice part of using image tiles is that, you don't need to worry about the different sizes of the menu pads when setting up the tiles, which before usually would be a headache as images have fixed sizes.

The even better thing of using image tiles is that, they are separated from your menu content. If you look back at the first two examples of "Using images in menu" and "Using images in menu [2]" made prior to version 5.4.2, which need some special separator items in menu content to hold border images, the image tiles solution lets you keep a clearer and more maintenance-friendly menu content.
