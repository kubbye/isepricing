Menu G5 Examples: IE Filters

yxScripts.com

 

This is a demo of different IE filters applied to the menu pads and the menu items at different stages, in which:

  • The "Home" item shows the transition filter effect when mouseout.
  • The "Contact" item shows the transition filter effect when mousedown.
  • The three sub-menu items show the transition filter effect when mouseover and mouseout.
  • The "Game Scripts" sub-menu shows the transition filter effect when it opens and closes.
  • The "Tool Scripts" and "User Forum" sub-menus show the transition filter effect when they open.
  • The menu items on sub-menus show transition filter effect when mouseover.
